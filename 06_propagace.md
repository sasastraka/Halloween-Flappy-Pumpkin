# 06 Propagace a cílení

## Strategie zveřejnění
Projekt je publikován na platformě GitHub a hostován přes GitHub Pages. To umožňuje okamžitý přístup k hratelné verzi bez nutnosti instalace.

## Designový záměr
Vizuální stránka je navržena tak, aby byla okamžitě rozpoznatelná na screenshotech. Mechanika "easy to play, hard to master" motivuje hráče k sdílení svých rekordů na sociálních sítích.
