# Halloween-Flappy-Pumpkin

## Autoři
- Alexandr Straka

## Popis
Halloween Flappy Pumpkin je jednoduchá 2D hra inspirovaná hrou Flappy Bird, ale tato verze má Halloweenský motiv. Hráč ovládá dýni, která musí překonávat překážky a sbírat body. Cílem hry je dosáhnout co nejvyššího skóre.  

## Živý obsah
Projekt je stále ve vývoji. Aktuálně obsahuje základní koncept hry a návrh herního designu. V budoucnu bude doplněno:
- Pohyb dýně (Už je, letí automaticky dopředu) 
- Generování překážek (Hotovo)
- Ukazatel skóre (Hotovo, možná se poupraví)
- Grafika a zvuky (Ve vývoji)
