# Flying-Bird-Adventure

## Autoři
- Alexandr Straka

## Popis
Flying Bird Adventure je jednoduchá 2D hra inspirovaná hrou Flappy Bird. Hráč ovládá ptáka, který musí překonávat překážky a sbírat body. Cílem hry je dosáhnout co nejvyššího skóre.  

## Živý obsah
Projekt je stále ve vývoji. Aktuálně obsahuje základní koncept hry a návrh herního designu. V budoucnu bude doplněno:
- Pohyb ptáka
- Generování překážek
- Systém skóre
- Grafika a zvuky
